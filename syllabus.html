INFO 2220 – Client Side Web Development – Fall 2019

Class Location

BA 503

Class Times

Tuesday & Thursday - 11:00 AM – 12:15 PM

Course Description

Introduction to interactive web development using a client-side language like JavaScript. Basic programming concepts common to almost all programming languages form the basis of the course. Exercises are designed to enhance students' problem-solving techniques and analytical thinking skills.

Learning Objectives

By the end of the course the student will be able to:

  • Design and implement interactive web pages.
  • Explain and utilize a wide range of the features available in the JavaScript programming language.
  • Write complex interactive scripts using JavaScript.
  • Analyze programming requirements in order to understand what data and processes are involved in the system.
  • Design a modular approach to satisfy those requirements.
  • Organize program code in web pages to implement the design.
  • Verify that the results obtained satisfy the original requirements
  • Understand and implement JSON web sources.
  • Understand the underlying principles of a prototyped language and the prototyping chain.
